# react_test

React 프론트엔드(`client`)와 Express 백엔드(`server`)를 함께 실행하는 기본 템플릿입니다.

## 실행 방법

```bash
npm install
npm run dev
```

- 프론트엔드: `http://localhost:5173` (사용 중이면 다른 포트로 자동 변경)
- 백엔드: `http://localhost:4000`
- 헬스체크 API: `http://localhost:4000/api/health`

## 폴더 구조

- `client`: React + Vite 프론트엔드
- `server`: Express 백엔드
